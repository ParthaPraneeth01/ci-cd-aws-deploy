# CI/CD Pipeline for AWS Deployment 🚀

## 📌 Project Overview
This project demonstrates a complete CI/CD pipeline using GitHub Actions to automatically deploy a static website to AWS S3 and serve it via CloudFront.

---

## 🛠️ Technologies Used
- GitHub Actions
- AWS S3
- AWS CloudFront
- IAM (Access Control)
- HTML

---

## ⚙️ Workflow
1. Code pushed to GitHub
2. GitHub Actions workflow triggered
3. Files uploaded to S3 bucket
4. CloudFront cache invalidated
5. Website updated automatically
